
# 🌦️ Weather API Project

A simple and responsive Weather App built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time weather information using the OpenWeatherMap API.

---

## 🚀 Live Demo

🔗 **CodePen:**  https://codepen.io/vraj206919/pen/MYJrQBo


---

## ✨ Features

- 🔍 Search weather by city name
- 🌡️ Display temperature in Celsius
- 💧 Show humidity
- 🌬️ Show wind speed
- ☁️ Display weather condition
- ⚠️ Error handling for invalid city names
- 📱 Responsive UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- OpenWeatherMap API

## 🔑 API Used

**OpenWeatherMap API**

https://openweathermap.org/current

Example Request:

```javascript
https://api.openweathermap.org/data/2.5/weather?q=London&appid=YOUR_API_KEY&units=metric
```

---

## ⚙️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Weather-API-Project.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

4. Search for any city to view live weather information.
